# Investment-Research-Assistant
An AI-powered tool for querying SEC 10-K filings using natural language. You pick a ticker, ask a question, and the app finds the most relevant sections from the company's annual filing and passes them to Claude to generate a grounded, source-cited answer. Under the hood it uses a RAG (Retrieval-Augmented Generation) pipeline — filings are chunked, embedded, and stored in a local ChromaDB vector store, so at query time the right context is retrieved and handed off to the Claude API for reasoning. Live market data (price, P/E, market cap) is pulled from Alpha Vantage and displayed alongside the answers.
Built with Python, Streamlit, ChromaDB, sentence-transformers, and the Anthropic API. To run it yourself, clone the repo, install dependencies with pip install -r requirements.txt, add your API keys to a .env file (see .env.example), run the ingest pipeline with python -m rag.ingest_pipeline --tickers NVDA AAPL MSFT, then launch the app with streamlit run app/main.py.
